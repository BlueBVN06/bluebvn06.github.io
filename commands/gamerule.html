Công dụng
Điều chỉnh luật lệ của Map
Đối tượng có thể dùng Command
Player có OP
Chủ Map
Addon
/execute

Command Block
Cú pháp
/gamerule <rule: BoolGameRule> [value: Boolean]
<rule: BoolGameRule> là Id mang tính bật/tắt. Danh sách hiện thị khi ghi Command
[value: Boolean] là phần trạng trái. Danh sách:
true là bật
false là tắt
/gamerule pvp false — Player sẽ không đánh được nhau
/gamerule <rule: IntGameRule> [value: int]
<rule: IntGameRule> là Id mang tính dữ liệu số. Danh sách hiện thị khi ghi Command
[value: int] là dữ liệu số
/gamerule randomtickspeed 5 — tốc độ tự nhiên phát triển tăng lên 5
Lưu ý
Giới hạn [value: int]
functioncommandlimit là 10000
maxcommandchainlength là 2147483647
randomtickspeed là 4096
spawnradius là 128
