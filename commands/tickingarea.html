/tickingarea
Công dụng
Load Chunk (chỉ tính x z) khi ngoài tầm nhìn của Player
Đối tượng có thể dùng Command
Player có OP
Chủ Map
Addon
/execute
/function
/gametest
Command Block
Cú pháp
/tickingarea add <from: x y z> <to: x y z> [name: string]
add là tạo
<from: x y z> là tọa độ bắt đầu
<to: x y z> là tọa độ kết thúc
[name: string] là đặt id cho tickingarea
/tickingarea add 0 0 0 1 1 1 test1 — tạo vùng Load Chunk tên test1 chứa tọa độ 0 0 0 đến 1 1 1. Vì tọa độ 0 0 0 và 1 1 1 đều trong Chunk (0,0;15,15) nên sẽ Load 1 Chunk (0,0;15,15)
/tickingarea add 0 0 0 17 17 17 test2 — tạo vùng Load Chunk tên test2 chứa tọa độ 0 0 0 đến 17 17 17. Vì tọa độ 0 0 0 trong Chunk (0,0;15,15), tọa độ 17 17 17 trong Chunk (16,16;31,31) nên sẽ Load 4 Chunk gồm Chunk (0,0;15,15), Chunk (0,16;15,31), Chunk (16,0;31,15) và Chunk (16,16;31,31)
/tickingarea add circle <center: x y z> <radius: int> [name:string]
add là tạo
circle là hình tròn (nhưng trong Game nó là hình vuông)
<center: x y z> là tọa độ trung tâm
<radius: int> là bán kính
[name: string] là đặt id cho tickingarea
/tickingarea add circle 0 0 0 2 test3 — tạo vùng Load Chunk vuông tên test3 trung tâm là Chunk (0,0;15,15) vì có chứa 0 0 0 với bán kính là 2 Chunk. Tức là sẽ Load 25 Chunk với kích thước 5 x 5 Chunk
/tickingarea list [all-dimensions: AllDimensions]
list là danh sách tickingarea
[all-dimensions: AllDimensions] là tất cả chiều không gian
/tickingarea remove <name: string>
remove là bỏ
<name: string> là id của tickingarea
/tickingarea remove <position: x y z>
remove là bỏ
<position: x y z> là tọa độ tickingarea được sử dụng
/tickingarea remove 0 0 0 — bỏ tất cả tickingarea chứa Chunk (0,0;15,15) vì 0 0 0 thuộc Chunk đó
/tickingarea remove_all
remove_all là bỏ tất cả
Lưu ý
/tickingarea có tối đa 10 lần sử dụng
1 tickingarea có thể Load tối đa 100 Chunk. Suy ra là 25600 Block (chưa kể chiều cao y)
1 tickingarea circle có thể Load tối đa 9 x 9 Chunk, tức là 81 Chunk. Suy ra là 20736 Block (chưa kể chiều cao y)
