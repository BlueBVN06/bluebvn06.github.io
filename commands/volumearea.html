/volumearea
Công dụng
Giới hạn vùng Fog thông qua Addon
Đối tượng có thể dùng Command
Player có OP
Chủ Map
Addon
/execute
/function
/gametest
Command Block
Cú pháp
/volumearea add <identifier: string> <from: x y z> <to: x y z> [name: string]
add là tạo
<identifier: string> là id File volumearea bạn tạo qua Addon
<from: x y z> là tọa độ bắt đầu
<to: x y z> là tọa độ kết thúc
[name: string] là đặt id cho volumearea
/volumearea add custom:test 0 0 0 0 0 0 test - tạo giới hạn Fog custom:test dưới id là test trong khoảng 0 0 0 đến 0 0 0
/volumearea list [all-dimensions: VolumeAreaAllDimensions]
list là danh sách volumearea
[all-dimensions: VolumeAreaAllDimensions] là tất cả chiều không gian
/volumearea remove <name: string>
remove là bỏ
<name: string> là id của volumearea đã tạo
/volumearea remove test - bỏ test
/volumearea remove <position: x y z>
remove là bỏ
<position: x y z> là tọa độ volumearea đã tạo
/volumearea remove 0 0 0 - bỏ tất cả volumearea chứa Chunk (0,0;15,15) vì 0 0 0 thuộc Chunk đó
/volumearea remove_all
remove_all là bỏ tất cả
