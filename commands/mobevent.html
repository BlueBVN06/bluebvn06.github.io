/mobevent
Công dụng
Điều khiển Event lớn
Đối tượng có thể dùng Command
Player có OP
Chủ Map
Addon
/execute
/function
/gametest
Command Block
Cú pháp
/mobevent <event: MobEvent> [value: Boolean]
<event: MobEvent> là id Event lớn. Danh sách:
events_enabled — Event lớn hoạt động
minecraft:ender_dragon_event — Event hồi sinh Ender Dragon
minecraft:pillager_patrols_event — Event Raid khi Player có Bad Omen gần Village
minecraft:wandering_trader_event — Event Spawn Wandering Trader theo đợt
[value: Boolean] là phần trạng trái. Danh sách:
true là bật
false là tắt
/mobevent events_enabled false — mọi Event lớn sẽ bị tắt
Lưu ý
Hiện tại /mobevent bị lỗi vậy nên Comamnd này coi như không hoạt động
