Cấu trúc Command
Cú pháp
/ Mở đầu Command
Không có <...> hoặc [...] Bắt buộc ghi lại
<...> Không thể bỏ qua, phải ghi 1 giá trị
[...] Có thể bỏ qua, giá trị mặc định sẽ được sử dụng hoặc nếu không có giá trị mặc định thì công dụng của Command bị thay đổi
Kiểu dữ liệu
int Kiểu số nguyên như -1, 0, 1, ...
float Kiểu số thực như -1.1, 0.1, 1.2, ...
string Kiểu chuỗi kí tự như xin_chao hoặc hi, nếu có khoảng cách hoặc kí tự đặc biệt thì phải để trong "..." như "Xin chào" hoặc "§1Hg"
boolean Kiểu đúng là true hoặc sai là false
json JavaScript Object Notation (JSON) là một định dạng trao đổi dữ liệu nhẹ
